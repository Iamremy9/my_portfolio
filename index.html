About Me

I'm a front-end developer and cybersecurity specialist with over 1 year of experience building secure, responsive, and user-friendly web applications. My unique blend of creative development skills and security expertise allows me to build beautiful interfaces that are also resilient against modern cyber threats.

My journey began in web development, creating interactive user experiences and responsive designs. As I deepened my technical knowledge, I became fascinated with web application security and began specializing in penetration testing to identify and remediate vulnerabilities in web applications.

I believe security should be integrated from the ground up in the development process, not added as an afterthought. This philosophy drives my approach to every project I undertake.

When I'm not coding or hunting for vulnerabilities, you can find me contributing to open-source security projects, attending cybersecurity conferences, or mentoring aspiring developers on secure coding practices.
